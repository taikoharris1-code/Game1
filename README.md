# Snake

A minimal browser-based Snake game.

Run locally:

```
python3 -m http.server 8000
# open http://localhost:8000 in your browser
```

Controls:
- Arrow keys to move
- Space or Pause/Resume button to pause
- Restart button to restart the game

Files:
- `index.html` — page
- `style.css` — styles
- `game.js` — game logic
